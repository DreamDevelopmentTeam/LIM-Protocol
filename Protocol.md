# LIM通讯协议

> LAN Instant Messaging Protocol

此版本的LIM使用UDP广播进行通信。

各节点之间通过发送**二进制的格式化JSON字段**进行数据教会。

---

## 端口规范

此版本统一接口为**65000**（发送/接收数据均使用此端口）。

绑定案例：("0.0.0.0", 65000)

| 参数 | 值 |
| --- | --- |
| 端口 | 65000/UDP |
| 方式 | 广播/去中心化 |
| 验证 | 无 |

--- 

## 网络消息格式

一般地，正常的通信信息格式如下：

```json
{
  "nid" : 0,
  "type": 0,
  "name": "system",
  "msg": "hello"
}
```

| 描述 | 作用 |
| --- | --- |
| nid | 网络ID（Network ID） |
| type | 消息类型 |
| name | 用户名称/操作名称 |
| msg | 消息数据/操作参数 |
